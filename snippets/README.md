# snippets

再利用コード置き場です。

## 使い方

- `snippets/python/` にジャンル別のコードを追加する
- 解く問題で使いたいときは `main.py` / `answer.py` に必要部分だけ貼る
- 問題で作った汎用ロジックがあれば、解き終わった後にここへ戻す

## 例

- `snippets/python/bfs.py`
- `snippets/python/dsu.py`
- `snippets/python/segment_tree.py`
